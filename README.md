# Quoco-REST
A sample spring REST Application to retreive quotations of policies from multiple providers. Built to understand docker.

### Build (requires Java 8+)
```bash
mvn clean install
```

### Dockerize (requires Docker desktop)
```bash
docker-compose up --build
```
